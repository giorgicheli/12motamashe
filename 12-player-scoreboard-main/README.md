# ამბასადორი ბეთი - ლიდერბორდი

თამაშის ლიდერბორდის სისტემა ავტომატური ქულების მართვითა და ანიმაციებით.

## ფაილების სტრუქტურა

- `index.html` - მთავარი ლიდერბორდის გვერდი
- `admin.html` - ადმინისტრაციული პანელი ქულების მართვისთვის
- `script.js` - მთავარი JavaScript ფუნქციონალობა
- `admin-script.js` - ადმინ პანელის JavaScript
- `styles.css` - მთავარი სტილები
- `admin-styles.css` - ადმინ პანელის სტილები

## გამოყენება

### მთავარი ლიდერბორდი (index.html)
1. გახსენით `index.html` ბრაუზერში
2. გუნდები ავტომატურად ალაგებენ ქულის მიხედვით (ყველაზე მაღალიდან დაბალისკენ)
3. თუ ადმინ პანელიდან ქულები განახლდა, გამოჩნდება "Update Leaderboard" ღილაკი
4. დააკლიკეთ განახლების ღილაკს რომ ნახოთ ახალი შედეგები ანიმაციებით

### ადმინ პანელი (admin.html)
1. გახსენით `admin.html` ბრაუზერში
2. **გუნდების ძებნა**: გამოიყენეთ ძებნის ველი გუნდის მოსაძებნად
3. **დაწყობა**: აირჩიეთ დალაგების კრიტერიუმი (სახელი, ქულა, ID)
4. **ინდივიდუალური მართვა**:
   - შეცვალეთ ქულა ველში და დააჭირეთ "განახლება"
   - გამოიყენეთ +10/-10 ღილაკები სწრაფი ცვლილებისთვის
   - "გაწმენდა" ღილაკი ანუલებს გუნდის ქულას
5. **ჯგუფური მოქმედებები**:
   - შეიყვანეთ რიცხვი და დააჭირეთ "ყველას დამატება" ყველა გუნდისთვის დასამატებლად
   - "ყველასთვის დაყენება" ყველა გუნდისთვის ერთნაირი ქულის დასაყენებლად
   - "ყველას გაწმენდა" ყველა ქულის ნულზე დასაყენებლად

## ფუნქციები

### ავტომატური დალაგება
- გუნდები ყოველთვის ალაგებენ ქულის მიხედვით (მაღალიდან დაბალისკენ)
- პოზიციების ცვლილება ხდება ანიმაციით

### Local Storage
- ყველა ცვლილება ინახება ბრაუზერის Local Storage-ში
- მონაცემები რჩება ბრაუზერის გადატვირთვის შემდეგაც

### Real-time განახლებები
- მთავარი გვერდი ამოწმებს განახლებებს ყოველ 2 წამში
- განახლების ღილაკი ანიმაციით გამოჩნდება ახალი ცვლილებების შემთხვევაში

### ანიმაციები
- გუნდების პოზიციის ცვლილება ხდება მსუბუქი ანიმაციებით
- Staggered animations პოზიციების განახლებისას

## ტექნიკური დეტალები

- **ბრაუზერის თავსებადობა**: ყველა თანამედროვე ბრაუზერი
- **მონაცემების შენახვა**: HTML5 LocalStorage
- **ანიმაციები**: CSS3 + Web Animations API
- **რესპონსიული დიზაინი**: მობილურ მოწყობილობებზე ადაპტირებული

## შენიშვნები

- მონაცემები ინახება მხოლოდ კონკრეტულ ბრაუზერში
- სხვა კომპიუტერზე ან ბრაუზერში მონაცემები არ იქნება ხელმისაწვდომი
- ბრაუზერის Cache-ის გასუფთავება შეიძლება წაშალოს მონაცემები
